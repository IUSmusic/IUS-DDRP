# Concept BOM — IUS DRRP Desktop

## Core compute and audio

| Item | Proposed part | Purpose |
|---|---|---|
| SBC | Raspberry Pi 4 (2GB or 4GB) | Main compute, UI, local playback/recording engine |
| Audio HAT | HiFiBerry DAC+ ADC | Stereo playback and stereo capture |
| Storage | microSD card | OS and local library |
| Display | Main OLED / TFT display assembly | EQ graph, radio, DAW overlay, status |

## Audio I/O and amplification

| Item | Proposed part | Purpose |
|---|---|---|
| Headphone amplifier | Dedicated low-noise headphone amp stage | Independent headphone output |
| Speaker amp | Compact class-D stereo amplifier | Internal speakers |
| Side speakers | Stereo full-range drivers | Main desktop listening |
| Top-firing drivers | Compact spatial/tweeter drivers | Spatial projection concept |
| Line input | Stereo line in jack / interface | External recording source |
| Line output | Stereo line out | External monitors / amp |
| XLR concept | Optional balanced input daughterboard | Future pro I/O expansion |

## Radio and control

| Item | Proposed part | Purpose |
|---|---|---|
| FM/DAB tuner module | To be selected by region and certification needs | Offline radio path |
| Main control knob | Rotary encoder / potentiometer | Volume / tune / navigation |
| 15-band EQ surface | Slider array + scanning electronics | Physical EQ and mixer-trigger control |
| Transport buttons | Tactile switch array | Playback and mode control |

## Power

| Item | Proposed part | Purpose |
|---|---|---|
| External PSU | Mains adapter / internal PSU stage | Primary power |
| Backup battery | Internal pack sized for ~1 hour under heavy use | Short-term backup |
| Power management | Charging / switchover / regulation board | Rail stability and failover |
