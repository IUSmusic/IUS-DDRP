# IUS DRRP Desktop — characteristics

## Product identity

**Official name:** IUS DRRP Desktop  
**Category:** Desktop hi-fi recorder / playback / offline radio workstation  
**Primary use case:** Studio and home recording, monitoring, playback, and offline radio listening.

## Core characteristics

- Mains-powered desktop unit with 1-hour internal backup battery under heavy use.
- Upgraded audio path built around the HiFiBerry DAC+ ADC HAT.
- Integrated side speakers plus top-firing spatial speaker concept.
- Dedicated line input, line output, headphones, USB-C, SD, and XLR visual I/O concept.
- 15-band physical EQ surface.
- Main display shows EQ graphs by default.
- When EQ sliders are moved, the UI switches to a 5-track DAW-style overview for recording, playback, and monitoring.
- Offline FM/DAB radio mode.
- Hi-res local playback and recording target: FLAC / 24-bit / 96 kHz concept.

## UX characteristics

- Low-latency tactile control.
- Immediate mode switching between player, radio, and record.
- Transparent visual feedback for input source, quality, elapsed time, and track/radio state.
- Studio-style manual EQ instead of menu-only EQ control.
